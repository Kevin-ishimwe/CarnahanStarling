\textbf{Derivation} \textbf{of the Carnahan--Starling Hard Sphere
Reference Term for SAFT Equation of State}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\item
  \textbf{Deriving the Carnahan-Starling Reference term}
\end{enumerate}

In the SAFT equation of state, the repulsive interactions in the segment
term are defined through a reference contribution is based on a
hard-sphere fluid. The Carnahan--Starling equation is used to describe
this reference HS contribution, as a function of the packing density
(\(\eta\)).

The compressibility factor for a CS reference terms is defined as:

\emph{\textbf{Main function -- CS compressibility factor}}

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.0522}}@{}}
\toprule()
\begin{minipage}[b]{\linewidth}\raggedright
\[Z^{CS} = \frac{1 + \ \eta + \ \eta^{2} - \eta^{3}\ }{(1 - \ \eta)^{3}}\]
\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(1)
\end{minipage} \\
\midrule()
\endhead
\bottomrule()
\end{longtable}

The Helmholtz free energy is related to the compressibility factor, as:

\emph{\textbf{Main function -- relating A to Z}}

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.0522}}@{}}
\toprule()
\begin{minipage}[b]{\linewidth}\raggedright
\[\frac{A^{res}(ref)}{Nk_{B}T} = \ \ \int_{0}^{\rho}{\frac{Z - 1}{\rho}\ d\rho}\]
\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(2)
\end{minipage} \\
\midrule()
\endhead
\bottomrule()
\end{longtable}

\textbf{Substituting} \textbf{Eq. 1} into \textbf{Eq. 2}, gives:

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.0522}}@{}}
\toprule()
\begin{minipage}[b]{\linewidth}\raggedright
\[\frac{A^{res}(ref)}{Nk_{B}T} = \ \int_{0}^{\rho}{\frac{\frac{1 + \ \eta + \ \eta^{2} - \eta^{3}\ }{(1 - \ \eta)^{3}} - 1}{\rho}\ d\rho}\ \]
\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(3)
\end{minipage} \\
\midrule()
\endhead
\bottomrule()
\end{longtable}

\textbf{Simplifying} the integer in the deviation:

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.0522}}@{}}
\toprule()
\begin{minipage}[b]{\linewidth}\raggedright
\[\frac{1 + \ \eta + \ \eta^{2} - \eta^{3}\ }{(1 - \ \eta)^{3}} - 1\  = \ \frac{1 + \ \eta + \ \eta^{2} - \eta^{3} - (1 - \ \eta)^{3}\ }{(1 - \ \eta)^{3}} = \frac{1 + \ \eta + \ \eta^{2} - \eta^{3} - (1 - \ 3\eta + 3\eta^{2} - \eta^{3})\ }{(1 - \ \eta)^{3}} = \frac{4\eta - \ {2\eta}^{2}\ }{(1 - \ \eta)^{3}}\]
\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(4)
\end{minipage} \\
\midrule()
\endhead
\bottomrule()
\end{longtable}

\textbf{\emph{Main relation}:} Since \(\eta\  \propto \rho\), then
\(d\rho/\rho\  \propto d\eta/\eta\) (\textbf{R.1})

\textbf{Substituting R.1} and \textbf{Eq. 4} into \textbf{Eq. 3}, gives:

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.0522}}@{}}
\toprule()
\begin{minipage}[b]{\linewidth}\raggedright
\[\frac{A^{res}(ref)}{Nk_{B}T} = \ \ \int_{}^{}{\frac{\frac{4\eta - \ {2\eta}^{2}\ }{(1 - \ \eta)^{3}}}{\eta}\ d\eta}\]
\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(5)
\end{minipage} \\
\midrule()
\endhead
\bottomrule()
\end{longtable}

\textbf{Simplify,}

\begin{longtable}[]{@{}
  >{\raggedright\arraybackslash}p{(\columnwidth - 2\tabcolsep) * \real{0.9478}}

\toprule()
\begin{minipage}[b]{\linewidth}\raggedright

\end{minipage} & \begin{minipage}[b]{\linewidth}\raggedright
(6)
